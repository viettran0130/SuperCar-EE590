Elma Documentation

This is final project for EE590: Advanced Programming for Embedded Systems. Using Event Loop Manager called elma to improve and extend functionalities car. These following functions will be implemented:

  • Engine: Finite State Machine was implemented to turn on and off the engine
  • Gear will contain 4 modes:
    • N = Neutral Mode
    • P = Park Mode
    • D = Drive Mode
    • R = Reverse Mode
  • Break
  • driver simulator

    Run the project

```bash git clone https://github.com/viettran0130/SuperCar-EE590.git cd SuperCar-EE590 docker run -v $PWD:/source -it klavins/elma:latest bash make cd examples bin/driving ```
